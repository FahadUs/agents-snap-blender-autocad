# Lessons for autocad_007 (Export & Headless) — read before every job

- [L010] If the installed AutoCAD is an **Education / Non-Commercial** license, warn the user that exported DWG/DXF/PDF carry an educational plot stamp (visible in commercial AutoCAD) — not for commercial deliverables. (source: AutoCAD 2026 Education)
- [L010+] **License scope:** Education is FUNCTIONALLY FULL (all features work). Limits are only: (1) educational stamp on anything saved/plotted THROUGH AutoCAD; (2) stamp CONTAGION — an Education file opened in commercial AutoCAD makes that copy stamp too; (3) legal: educational use only. **Clean-output rule:** **ezdxf headless does NOT pass through AutoCAD → its DXF is CLEAN (no stamp).** COM/live AutoCAD output carries the stamp. → For a clean 2D deliverable, choose the ezdxf path; reserve COM for work needing live AutoCAD, and warn about the stamp. Serious commercial production needs a commercial license (or BricsCAD/ZWCAD).
- [COM] On full AutoCAD, native DWG export is available via COM `doc.SaveAs(path)` (ezdxf can only write DXF). Prefer DWG when the consumer uses AutoCAD. (source: live COM run)
- [L021] Before taking the verification screenshot, close/hide ALL palettes — SendCommand `_PROPERTIESCLOSE` + `_TOOLPALETTESCLOSE`, dismiss the Connect/sign-in panel, optionally `CLEANSCREENON` — then ZoomExtents and capture. Confirm the captured PNG shows the WHOLE footprint with no overlay before declaring verification done; closing only the Properties palette is NOT enough. (source: docked palette occluded BEDROOM 2/DRESSING/BATH 1 in villa_plan_acad.png)
